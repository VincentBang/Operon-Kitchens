# Lead First Response Script

Last updated: 17 June 2026

Purpose: provide safe first-response wording for controlled request-review leads.

## General Reply

```text
Hi [Name], thanks for sending your kitchen project details through Operon Kitchens.

We will review the information you provided and identify the next useful step. At this stage, any estimate or review is planning guidance only. Project-specific pricing requires site measure, confirmed selections, licensed trade review where relevant and written scope confirmation.
```

## If They Have A Quote

```text
If you already have a kitchen quote, the most useful next step is usually to check whether the scope, inclusions, exclusions, PC sums, provisional sums, appliance allowances, benchtop/splashback details and trade items are clear before comparing totals.
```

## If They Are Early Stage

```text
If you are still planning, we can start with a planning range and confidence review, then identify what details would improve the estimate before site measure.
```

## If Apartment Or Strata

```text
Apartment and strata kitchens may need extra review for access, lift/parking, work hours, owners corporation requirements, licensed trades and site conditions. These are review prompts only and not legal or strata approval.
```

## If Site Measure Is Needed

```text
Based on the information supplied, a site measure may be the safest next step before relying on project-specific pricing. The measure helps confirm dimensions, access, services, site conditions, selections and written scope.
```

## Do Not Say

- final quote
- guaranteed saving
- approved
- compliant
- legally checked
- certified
- order now
- we can beat the quote
